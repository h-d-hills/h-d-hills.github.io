web projects

I am, in a loose sense of the term, a front end web developer. here are some of the webpages I have made:

simple layout


this is a photo gallery project. it has a fairly simple layout, but it works. for this project, I familiarized myself with javascript plugins. images are highlighted as you mouse over them and expand when clicked. a search bar at the top allows you to search for specific images

complex layout


I don't know exactly how long this project took me. the days seemed to blend together and it felt like two weeks. however, I am very pleased with the result. I had to draw upon my knowledge of html, scss, javascript and svg images to pull this off. there are plenty of graphs, which are always good things to have

web game


for this project, I applied my understanding of javascript to developing a guessing game with a health system. the name of this game is Victory Big Success Time (Talking Heads Edition). Talking Heads is my favorite band of all time, so I saw it fitting to make this project about them. functionally this is identical to hangman, but stylistically it resembles a typing simulator

api implementation


this project is similar to the photo gallery project, except the data here are different every time you load the page. I used an online api to pull false user ids from an online fake people generator and displays the data in a user-friendly fashion
